Система управления идеями — это веб-приложение для сбора, обсуждения и голосования за предложения по улучшению компании. Система позволяет сотрудникам предлагать идеи, голосовать за них и комментировать, а администраторам — модерировать контент и управлять пользователями.

<h1>Доступ администратора</h1>
<h2>Первый запуск системы</h2>
При первом запуске проекта будет доступен только один пользователь — администратор:

Логин: 1QsMeP23
Пароль: 12345

<h3>Примечание: </h3>
Логин администратора можно изменить в файле backend/database.py, отредактировав строку кода с "username": "1QsMeP23".

<h2>Первый вход администратора</h2>
Когда администратор входит в систему в первый раз, ему будет предложено изменить пароль на собственный. После смены пароль перезаписывается в файле users.json в хешированном виде.

<h1>Управление пользователями</h1>
<h2>Создание новых пользователей</h2>
Только администратор может создавать новых пользователей. При массовой генерации сотрудников:

1.Система генерирует случайные логины и пароли

2.Пароли показываются администратору в модальном окне

3.Администратор может:

Скопировать логины и пароли для раздачи сотрудникам
Сохранить данные в CSV файл
Подтвердить сохранение

<h1>Безопасность паролей</h1>
<h2>После сохранения паролей: </h2>

Пароли хешируются и сохраняются в users.json
Администратор не сможет узнать пароль сотрудника после сохранения
Сотрудники при первом входе также могут сменить пароль

Запуск проекта через Docker: 

Перед началом не забудьте запустить приложение Docker
1. Перейдите в папку ../workProj и в консоли введите docker-compose up -d
2. После этого в браузере откройте http://localhost
3. Для остановки введите docker-compose down

Вам надо будет поменять текст "Название компании" на реальное название, которое вы захотите для этого надо:
1. Верхняя шапка. Перейдите в файл frontend/my-react-app/src/App.js и найдите <h1>Название компании</h1>
2. Нижний футер. Перейдите в файл frontend/my-react-app/src/App.js и найдите там function Footer()

Если вы хотите поменять цвет, то в файле variables.css прописаны какие свойства за что отвечают.
